# Setup

## Requirements

- Node.js 20 or newer.
- Package manager: npm.

## Install

`npm install`

## Quick Start

`npm run dev -- --root . update`

## Development

`npm run dev`

## Build

`npm run build`

## Test

`npm run test`

## Freshness Check

`npm run dev -- check`
